Welcome to the Bulk Search & Replace tool for WordPress CLI users!

This tool is built for bulk search & replaces in mind, where WP CLI is handier than any generic plugin's Search & Replace, and can be far easier, albeit indimidating to some. The tool should provide a concise output to run on an SSH terminal that will allow you to fix a multitide of https based errors.

How to use this tool:

  • In the Boxes below, insert the domains present on your mixed content errors (you can also use a tool like WhyNoPadlock to scan your site to give you a log of those errors too.

  • Now in the box below, input the primary domain you are using on Flywheel (this is the domain that you have clicked "Mark as Primary". If you arent sure on what you have selected; check your dashboard or check out our Going Live Documentation

  • When this is all said and done, at the very bottom; you should have a WP CLI command you can copy and paste into a terminal, that you have an active SSH connection to your WordPress site.
